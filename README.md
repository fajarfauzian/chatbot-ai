## Fitur

- Interface chat yang responsif
- Riwayat percakapan sementara
- Integrasi dengan API AI
- Desain modern dengan Bootstrap

## Persyaratan

- Node.js (versi 14 atau lebih tinggi)
- npm
- API key dari OpenAI
